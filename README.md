# 💼 Online Job Finder

> **BCA Final Year Project (6th Semester)**  
> A web-based Online Job Finder platform developed using **ASP.NET (C#)** and **SQL Server** that connects job seekers with employers through an efficient recruitment system.

---

## 📖 Overview

Online Job Finder is a recruitment platform designed to simplify the hiring process by providing separate dashboards for **Job Seekers**, **Employers**, and **Administrators**.

The system enables job seekers to search and apply for jobs, employers to post and manage vacancies, and administrators to manage users and job postings.

---

## 🚀 Features

### 👤 Job Seeker

- Create and update profile
- Upload resume
- Search available jobs
- Apply for jobs
- Track application status

### 🏢 Employer

- Register employer account
- Post job opportunities
- Manage job listings
- View applicants
- Shortlist candidates

### 🔑 Admin

- Manage users
- Approve or reject job postings
- Monitor platform activities
- Generate reports

## ⚙️ Getting Started

### Prerequisites

- Visual Studio 2019 or later
- SQL Server
- .NET Framework

### Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/OnlineJobFinder.git
```

2. Open the project in Visual Studio.

3. Restore the SQL Server database.

4. Update the connection string inside **Web.config**.

5. Run the application.
